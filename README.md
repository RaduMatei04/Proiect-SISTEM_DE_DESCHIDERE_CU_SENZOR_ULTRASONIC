# Proiect-SISTEM_DE_DESCHIDERE_CU_SENZOR_ULTRASONIC
Acest proiect a fost realizat in cadrul Sesiunii de Comunicari Stiintifice UPB din Mai 2024. Functionalitatea consta in controlarea unui servomotor, care este activat cu ajutorul datelor receptionate de un senzor ultrasonic de distanta. A fost folosit un microcontroller ESP32 iar codul a fost realizat in Arduino.
